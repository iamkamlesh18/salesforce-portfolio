# Contributing

Thanks for your interest in contributing to this project! Some quick guidelines:

- Fork the repository and create a feature branch for your changes.
- Run `npm ci` and ensure `npm run prettier`, `npm run lint`, and `npm test` pass locally before opening a PR.
- Keep changes focused and include tests where appropriate.
- Use clear commit messages and reference related issues in PR descriptions.

We welcome improvements, bug fixes, and documentation updates — thanks for contributing! ❤️
